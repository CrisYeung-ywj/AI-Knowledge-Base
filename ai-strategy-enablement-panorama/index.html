05 Agent Capability

一个超级 Agent = 有记忆的数字员工

有大脑会规划、有小脑快执行、有手脚能操作电脑和工具，还能记住偏好、上下文与历史任务；一个机器人连起钉钉、本地电脑、业务系统与知识库。
能力示意大脑 + 手脚 + 记忆

🧠 大脑

理解目标，判断复杂度，拆解步骤，决定先查什么、先做什么、产出什么。

⚡ 小脑

把明确动作快速执行：读取资料、整理表格、生成报告、跑脚本、批量处理。

🦾 手脚

操作电脑、文件、浏览器和业务系统，完成从“想法”到“附件交付”的最后一步。

大脑｜规划
小脑｜快执行
手脚｜操作电脑
工具｜调用系统
知识库｜有依据
记忆｜越用越懂
林氏小龙虾 Agent

🔧 工具

按任务调用钉钉、本地电脑、表格、浏览器、代码能力和市面成熟工具。

📚 知识库

连接制度、SOP、产品资料、客服话术、经营口径，让回答和交付有依据。

🧩 记忆

记住用户偏好、历史任务、常用模板、部门案例和上下文，越用越贴近真实工作。

一个超级 Agent + 工具链 + 记忆：能想、能做、能连，也能把经验沉淀下来
